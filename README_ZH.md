# Babytouch

Babytouch 是一款专为幼儿设计的照片和视频浏览器。它采用了类似 TikTok 的交互界面，并通过禁用“返回”键和其他 UI 控件来防止意外触碰，让孩子们可以通过简单的垂直滑动安全地探索内容。

## ✨ 功能特点

- 📱 **沉浸式全屏体验** – 去除所有干扰元素，让宝宝完全专注于内容。
- 🎬 **TikTok 风格滑动** – 通过直观的上下滑动，轻松浏览照片和视频。
- 📹 **智能自动播放** – 视频在出现在屏幕上时自动播放，滑走时立即暂停。
- 🔒 **幼儿防误触设计** – 禁用返回键和系统导航，防止意外退出或误操作。
- 📂 **灵活的相册选择** – 支持选择特定相册，并包含“所有视频”快捷方式以便快速浏览。
- 🔊 **优化的音频体验** – 视频默认以 50% 的音量播放，确保舒适且安全的听觉体验。
- 🌍 **多语言支持** – 支持英语、简体中文、繁体中文、日语和韩语。
